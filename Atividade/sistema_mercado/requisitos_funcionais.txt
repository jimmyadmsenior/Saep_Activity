OS 12 REQUISITOS FUNCIONAIS MAIS IMPORTANTES
================================================================================

RF001 - O sistema deve permitir que usuários façam login com usuário e senha
RF002 - O sistema deve validar credenciais no banco de dados usando prepared statements
RF003 - O sistema deve armazenar senhas criptografadas usando password_hash()
RF004 - O sistema deve permitir cadastro de produtos com nome, código de barras, categoria, marca, preços e estoques
RF005 - O sistema deve permitir listagem de todos os produtos
RF006 - O sistema deve permitir edição de dados de produtos existentes
RF007 - O sistema deve calcular e exibir status do estoque (OK, Baixo, Esgotado)
RF008 - O sistema deve registrar movimentações de entrada e saída de estoque
RF009 - O sistema deve atualizar automaticamente estoque após movimentações
RF010 - O sistema deve exibir dashboard com estatísticas de produtos e estoque
RF011 - O sistema deve listar produtos ordenados alfabeticamente usando algoritmo Bubble Sort
RF012 - O sistema deve proteger todas as páginas internas com verificação de autenticação
